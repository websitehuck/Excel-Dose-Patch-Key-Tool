  "openai_api_key": "sk-... (or use env var OPENAI_API_KEY)",

  "responsive_ui": true,
  "enable_collab": false,

```bash

dose-excel --config ~/dose_excel_config.json
```

## Console Invocation Example 💻

Fire up **Dose for Excel 2026** from the command line with a one-liner transformation:

```bash
dose-excel --input ./sales_q4_2025.xlsx \
           --unpivot months:jan,feb,mar \
           --rename column:total_revenue -> net_revenue \
           --apply-formula "IF(C2>5000, 'High', 'Standard')" \
           --export ./output/sales_clean.xlsx \
           --audit
```

Expected output:

```
✓ File loaded: sales_q4_2025.xlsx (12,400 rows)
✓ Unpivot executed: 3 columns → 37,200 normalized rows
✓ Formula applied to column D
✓ Audit trail saved to: ./output/audit_2026-01-15.log
✓ Export complete (2.1s)
```

Pro tip: Combine with cron/scheduled tasks for nightly data refreshes.
